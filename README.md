# MyWebComicReader

> A very simple web based .zip comic reader.
>
> 一个非常简单的基于网页的.zip压缩包漫画阅读器。

加载多图压缩包时会很慢。

写的很low，随缘更新  ~~(菜)~~。

功能比较简单，只能看zip压缩包的漫画，只要里面是漫画，套了几层 ~~buff~~ 文件夹都行。

文件类型的判断也还没整，之后随缘更。

[**DEMO**](https://monsterhhe.github.io/MyWebComicReader)
